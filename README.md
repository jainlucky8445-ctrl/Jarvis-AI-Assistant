# 🤖 Jarvis AI Assistant

An intelligent voice-controlled AI assistant built with Python. Jarvis can listen to voice commands, answer questions using AI, open applications, remember notes, search the web, provide entertainment features, and perform various daily tasks.

---

## ✨ Features

- 🎤 Voice Command Recognition
- 🧠 AI-powered Conversations (Groq API)
- 🗣️ Text-to-Speech Responses
- 📂 Open Desktop Applications
- 🌐 Open Websites
- 📝 Memory System (Save & Recall Notes)
- 📰 Wikipedia Search
- 🕒 Tell Current Time
- 🎲 Flip Coin & Roll Dice
- 🎬 Movie Recommendations
- 🏏 Cricket Score
- ⚽ Football Score
- 💡 Daily Quotes
- 🔊 Wake Word Ready Architecture
- 🔧 Modular Python Project Structure

---

## 🛠️ Tech Stack

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/jainlucky8445-ctrl/Jarvis-AI-Assistant.git
```

Move into the project

```bash
cd Jarvis-AI-Assistant
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file.

```
GROQ_API_KEY=YOUR_API_KEY
```

---

## ▶️ Run

```bash
python main.py
```
